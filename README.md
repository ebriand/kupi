# kupi

## How to flash a sd card

1. Put the sd card into your reader
1. Download the hypriot os here: https://github.com/hypriot/image-builder-rpi/releases/
1. Replace the xxxx in the user-data.yaml file with your wifi ssid and password
1. Edit the flash.sh to set the name of your os file and execute the flash.sh script
1. Boot up the rpi
1. Enjoy :)
